# .Simple-Chatbot
# 🤖 Simple Chatbot Project

## 🔍 Overview

This project is a simple rule-based chatbot developed using Python. It interacts with users by responding to predefined inputs and demonstrates basic conversational logic.

## 🛠️ Technologies Used

* Python

## 📈 Features

* Takes user input from console
* Responds using conditional logic (if-elif statements)
* Handles basic conversation like greetings and queries
* Exit option to end chat

## 💻 How It Works

The chatbot uses simple `if-elif` conditions to match user input and return appropriate responses.

## 🚀 How to Run

1. Install Python
2. Copy the code into a `.py` file
3. Run the program:

   ```
   python chatbot.py
   ```

## 📁 Project Structure

* chatbot.py → Main chatbot code

## 💡 Example Conversation

```
You: hello
Bot: Hello! How can I help you?

You: how are you
Bot: I am fine, thank you!

You: bye
Bot: Goodbye!
```

## 🎯 Conclusion

This project demonstrates basic Python programming concepts like loops, conditionals, and user interaction.
